### Research Questions
- Which musical features are associated with higher chart rankings?
- How have musical characteristics changed over time?
- Can chart success be predicted using audio features?
### Methods
- Data cleaning and preprocessing in R
- Exploratory data analysis with ggplot2
- Linear regression and Random Forest models
- Model evaluation using RMSE
### Key Findings
- Danceability and loudness are the strongest predictors of chart success
- Popular music has become less acoustic over time
- Musical features alone cannot fully explain chart success
